> **Minimalist Productivity Meets Aesthetic Design.**
> FocusFlow is a smart task management PWA built for deep focus, featuring a beautiful "Pastel Dreams" Glassmorphism UI.

---

## ✨ Features

* **⚡ Progressive Tasks**: Beyond binary "done/undone." Cycle through **0% to 100%** completion.
* **⏲️ Pomodoro Focus Timer**: Integrated timer to help you maintain a "Flow" state.
* **📱 PWA Ready**: Install FocusFlow directly to your iOS/Android home screen.
* **🎨 Glassmorphism UI**: A lightweight, modern aesthetic with pastel gradients.
* **💾 Persistent Storage**: Automatic synchronization with `LocalStorage`.
